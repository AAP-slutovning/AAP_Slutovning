# AAP_Slutovning

##  Sprintmål - Fungerande meny där man kan lägga till, ta bort och se uppgifter.

## Sprint backlog:
1. Som användare vill jag ha en tydlig meny för att enkelt navigera i applikationen (Lena).
2. Som användare vill jag kunna lägga till en uppgift för att hålla koll på vad jag ska göra (Mikael).
3. Som användare vill jag kunna se alla mina uppgifter för att få en överblick (Karin).
4. Som användare vill jag kunna ta bort en uppgift för att kunna rensa i min lista (Tomas).

### Vid mån av tid:
5. Som användare vill jag kunna markera en uppgift som klar för att veta vad jag är färdig med.
6. Som användare vill jag kunna filtrera på klara/oklara uppgifter för att fokusera på rätt saker.
